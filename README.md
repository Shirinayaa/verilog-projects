# Verilog Projects

## Overview

This repository contains my Verilog RTL projects built while learning digital design and VLSI fundamentals.

The goal of this repository is to move beyond basic modules and build larger system-level designs by combining multiple concepts like combinational logic, sequential logic, finite state machines, memory, and communication protocols.
